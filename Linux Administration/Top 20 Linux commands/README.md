# Top 20 Linux Commands Every DevOps Engineer Should Know

## 📌 Overview

Linux is the backbone of modern Cloud and DevOps environments.

Whether you're managing AWS EC2 instances, Kubernetes worker nodes, Docker hosts, or CI/CD servers, Linux is used everywhere.

This repository contains the **Top 20 Linux Commands** that every DevOps Engineer should know before attending interviews or working on production systems.

---

## 📚 Commands Covered

| Command | Purpose |
|----------|---------|
| ls | List files and directories |
| cd | Change directory |
| pwd | Print current directory |
| cat | Display file contents |
| less | View large files |
| cp | Copy files |
| mv | Move or rename files |
| rm | Remove files |
| mkdir | Create directory |
| rmdir | Remove empty directory |
| touch | Create empty file |
| chmod | Change permissions |
| chown | Change ownership |
| ps | View running processes |
| top | Monitor CPU and Memory |
| kill | Stop a process |
| df -h | Check disk usage |
| du -sh | Check directory size |
| grep | Search text |
| find | Search files |
